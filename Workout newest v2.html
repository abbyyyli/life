本週完成狀態

—

今日課表

照著做即可。重量、組數可選填；如果你用 Hevy 記錄，這裡可以只按完成或留空。

Actual Workout｜實際完成

臨時換動作時，把實際做的動作加在這裡；它會進入 History / Progress。

Current Week｜本週課表

每週可以在這裡用選單調整課表；Templates 是原始課表庫，不需要一直覆蓋。

Workout Templates

這是預設課表庫。之後每週調整時，主要改 Current Week。

Exercise Library

精簡版，只收你目前器材與目標真正會用到的動作。每張卡都要能看懂「這是練哪裡、怎麼做、錯在哪」。

新增動作 JSON 匯入

未來遇到新動作，把動作介紹丟給 AI 生成 JSON，再貼到這裡加入 Exercise Library。
新增動作 Prompt（短版）

Workout Import

可貼 Hevy 轉出的 JSON，或把 Hevy 截圖丟給 AI 後貼回來。
Hevy / Workout Parser Prompt

Program / Current Week Import

貼上我幫你輸出的 Program JSON，可更新本週課表，也可以一起更新 Templates 與 Exercise Library。這是「課表匯入」，不是訓練紀錄匯入。
Program Import JSON 格式

Progress

只看必要資訊：完成率、訓練天數、肌群頻率、常見重量。

History｜歷史訓練

Coach Notes

記錄做不了、太難、疼痛、沒感覺。這是每週調整課表的依據。

    Guide & Prompts

    這裡是可編輯 Prompt Library。內容會自動儲存在 Workout OS，並跟全部備份一起匯出。

    ✏️ Direct Prompt Editor｜直接編輯 Prompt

    選一段 Prompt 後直接修改；按「套用」會同步更新下方相同 Prompt、localStorage，並跟全部備份一起匯出。這個區塊是給 GitHub Pages 使用時更穩定的直接編輯入口。

    📖 Project Intro｜專案介紹

    未來開新對話時，可以先貼這段，讓 AI 知道 Workout OS 的定位與資料結構。

    🧱 JSON Robustness Rules

    所有 Workout prompt 共用的輸出規則，避免 AI 輸出壞掉。

    🏋️ Program Generator｜生成下一週課表

    根據器材、目標、History、Progress、Coach Notes 生成可匯入的 Program JSON。

    🔄 Program Import Prompt｜可匯入課表格式

    讓 AI 直接輸出可貼進 Program / Current Week Import 的 JSON。

    📅 Current Week Generator｜只更新本週

    只想換本週安排、不新增課表模板時使用。

    🧩 Template Generator｜新增課表模板

    新增臀腿A-2、肩背C、Core C 等課表變化版本。

    🏃 Workout Parser｜訓練紀錄轉 JSON

    把 Hevy 截圖或自然語言訓練紀錄轉成 Workout Log JSON。

    📚 Exercise Generator｜新增動作資料

    新增 Exercise Library 動作時使用。

    📊 Weekly Review｜週回顧與調整

    每週把 History、Progress、Coach Notes 貼給 AI 後，請它分析並輸出下週 Program JSON。

    📥 Program Import Guide｜匯入格式說明

    說明 Program Import 可以接受哪些 JSON，以及會覆蓋哪些資料。

    🤖 AI Coach｜即時調整課表

    今天時間不夠、太累、器材受限或局部痠痛時，請 AI 幫你臨時調整課表。

    設定

    Workout Coach OS 是個人訓練輔助工具，不是醫療或復健建議；疼痛請停止並尋求專業協助。
